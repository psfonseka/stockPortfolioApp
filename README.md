# Stock Portfolio App

Spotify Challenge

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running

```sh
npm start
```
Then to localhost:3000
